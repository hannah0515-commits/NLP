# NLP

This dataset is the Customer Support posts from the biggest brands like ‘apple support’,  ‘amazon help’ , ‘uber support’ on Twitter. The data stored in a csv file, each row consists of tweet_id, author_id, inbound, create data, text etc. They are either new post or response to the exist post.

Using different preprocessing steps
• Tokenization
• Stop words removal
• Stemming
• Lemmatization
• And other pre-processing tasks
