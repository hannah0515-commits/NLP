Data Set
Movie reviews dataset will be used for this assignment and dataset can be downloaded from this link https://www.kaggle.com/c/word2vec-nlp-tutorial/data. 
labeled data set consists of 50,000 IMDB movie reviews, specially selected for sentiment analysis. The sentiment of reviews is binary, 
meaning the IMDB rating < 5 results in a sentiment score of 0, and rating >=7 have a sentiment score of 1. 
No individual movie has more than 30 reviews.
